# 口播网站 - AI 语音合成平台

## 项目概述
一个基于 AI 的口播内容生成平台，用户可上传文字和音频样本，生成高质量的语音内容。

## 技术架构
- **前端**: React + TypeScript + Web Audio API
- **后端**: Spring Boot + Spring Cloud (Java)
- **AI 引擎**: Coze Voice Gen API + 音频处理
- **数据库**: PostgreSQL + Redis
- **部署**: Docker + Kubernetes

## 盈利模式
1. **订阅制**: 月费 29-99 元，包含免费时长
2. **按量付费**: 每分钟 0.5-2 元（根据音色质量）
3. **企业 API**: 批量调用，按请求计费
4. **定制服务**: 专属音色训练 + 品牌定制
